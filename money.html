Capital sources columns are dual-purpose: for cash rows the columns mean Bank (live bank-feed balance, manually entered from QBO Banking) / Posted (GL balance from qbo_transactions mirror) / For Review (Bank − Posted, the QBO categorization backlog) / Available (Bank when set else Posted). For debt rows they retain Cap / Drawn / Available / Cost-when-drawn semantics. Bank-feed balance is not exposed via the QBO Accounting API — only GL CurrentBalance is — so the Bank column is populated by manual update of capital_sources.bank_balance_manual.
