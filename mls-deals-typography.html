3 · Numbers = DM Sans + tabular-nums · mono = codes only
Same rule money.html adopted. Today mls-deals has zero tabular-nums, so numeric columns drift. Add font-variant-numeric:tabular-nums to any container holding aligned numbers. JetBrains Mono stays only on MLS#/DB ids.
